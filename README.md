# Graficos WebGL
Proyecto Neon Night Drive de conducir un coche entre la ciudad en ambientacion Neon. 

## Nombres de los integrantes y de sus Usuarios GitHub
Luca Vella  -- MrLucaa1



## Objetivo del proyecto

Este proyecto tiene como objetivo crear un infinite runner de conduccion con distintos shaders del tiempo con ambiente Neon 

## Plan de Sprints

| 1 | Creacion de la idea del proyecto.
| 2 | Implamentacion de los objetos y Programacion de ls movimientos.
| 3 | Pulido de detalles y la presentacion.
